We've created a small sample database to use for this assignment. It contains four relations: 

Person(name, age, gender)       // name is a key
Frequents(name, pizzeria)       // [name,pizzeria] is a key
Eats(name, pizza)               // [name,pizza] is a key
Serves(pizzeria, pizza, price)  // [pizzeria,pizza] is a key
Instructions: You can execute the queries using Relational Algebra or SQL queries. Since Relational Algebra symbols aren't readily available on most of the keyboards, refer to this syntax guide. You can also view the database here.
